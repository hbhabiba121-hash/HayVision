# HayVision Platform

A Moroccan neighborhood and real estate analysis platform, designed to visualize property trends, predict future value, and provide insights for investors and citizens. Built with a modern React + TypeScript stack, responsive design, and scroll-triggered animations.

## Features

- **Preloader** with animated brand reveal and establishment year
- **Navigation** with desktop dropdowns and mobile menu with smooth transitions
- **Hero Section** with Ken Burns background, key statistics, and call-to-action buttons
- **Neighborhood Insights** with interactive charts, heatmaps, and feature highlights
- **Market Trends** with recent news, analysis articles, and testimonials
- **Contact Section** with Formspree integration, contact info cards, and success/error states
- **Newsletter Subscription** with Formspree integration
- **Footer** with social links, legal links, and back-to-top button
- **Scroll To Top** floating button
- All sections have scroll-triggered entrance animations (fade-up, slide-in-left/right, scale-in)
- Fully responsive design
- All content is driven by a single `config.ts` file

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
- shadcn/ui components
- Formspree (contact form / newsletter)
